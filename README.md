# Template: Alexa Audioplayer

# Build
    * jovo build --stage dev
    * jovo build --stage prod

# Deploy
    * jovo deploy --stage dev (Sets DEV Endpoint)
        * Use jovo run to start the local debugger
    * jovo deploy --stage prod (default, Sets PROD endpoint)
